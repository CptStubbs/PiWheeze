# PiWheeze
At home CO2 monitoring, connected to Apple Home

## This Project
This project assumes a few things:
1. A Raspberry Pi 3B+
2. A SCD30 CO2 + Temp + Humidity sensor connected over I2C

## How to run
Clone the project onto a compatible setup (Raspberry Pi + Sensor)

Then to run:
`docker compose up --build`
